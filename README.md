# search-techniques-demo

Small demo repository showing several search techniques:

- Linear search
- Binary search (on sorted arrays)
- Trie (prefix search)
- Inverted index (boolean OR/AND queries)
- Simple BM25-like scorer

Usage

Run examples:

```powershell
python example_runner.py
```

Run tests (requires pytest):

```powershell
pip install -r requirements.txt
pytest -q
```
